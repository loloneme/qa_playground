# Баг 1


| **Поле**                  | **Значение**                                                                                  |
| ------------------------- | --------------------------------------------------------------------------------------------- |
| **Название**              | Некорректно работает фильтрация От самых дорогих товаров к дешевым                            | 
| **Описание**              | При сортировке товаров От дорогих к дешевым (Дороже), товары находятся в некорректном порядке |     
| **Шаги**                  | 1. Перейти на вкладку с горными велосипедами.  <br>2. Выбрать сортировку "Дороже"             |     
| **Ожидаемый результат**   | Товары успешно отсортированы по убыванию цены                                                 |     
| **Фактический результат** | Товары не отсортированы корректно. Большая цена следует после меньшей                         |     
| **Окружение**             | Ноутбук ACER Aspire, Windows 11, GoogleChrome                                                 |     
| **Приоритет**             | Высокий                                                                                       |     
| **Серьезность**           | Значительная                                                                                  |

# Баг 3

# Баг 4

| **Поле**                  | **Значение**                                                                                                |
| ------------------------- | ----------------------------------------------------------------------------------------------------------- |
| **Название**              | Некорректная сортировка по выбранному региону                                                               |
| **Описание**              | При сортировке "Сначала из Москвы" отображаются объявления с некорректным местоположением                   |
| **Шаги**                  | 1. Перейти на вкладку с горными велосипедами.  <br>2. Выбрать сортировку "Сначала из Москвы"                |

| **Фактический результат** | Среди объявлений из выбранного региона (Москвы) отображаются объявления из других регионов                  |
