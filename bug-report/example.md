# Обучение составлению баг-репорта

Описание задания:
Вы — тестировщик, работающий над проектом интернет-магазина. Во время тестирования вы обнаружили, что при попытке добавить товар в корзину система выдает ошибку Bad request. Ваша задача — создать баг-репорт, чтобы сообщить о проблеме разработчикам. Используйте предоставленную информацию и структуру баг-репорта.

| **Поле**                  | **Значение**                                                                                                                                                                                                                                                           |
| ------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Название**              | Результат Bad Request при добавлении товара в корзину                                                                                                                                                                                                                                     |
| **Описание**              | При добавлении товара в корзину возвращается ошибка 400 Bad Request                                                                                                                                                                                                                 |  
| **Шаги**                  | 1. Перейти на вкладку с товарами.  <br>2.Выбрать товар <br>3. Нажать на кнопку "Добавить в корзину" <br>4. Нажать кнопку "Зарегистрироваться". |  
| **Ожидаемый результат**   | Товар успешно добавлен в корзину, и пользователь видит сообщение "Товар добавлен в корзину" |     
| **Фактический результат** | Товар не добавлен в корзину, после нажатия появляется сообщение "500 Bad Request"                                                                                                                                                                                     |     
| **Окружение** | Ноутбук ACER Aspire, Windows 11, GoogleChrome |
| **Приоритет**                |                   Высокий                                                    |     
| **Серьезность**             | Блокирующая           
|**Доп. информация**||
