## Регистрация нового пользователя

| **Поле**                  | **Значение**                                                                                                                                                                                                                                                           |
| ------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **ID**                    | TC001                                                                                                                                                                                                                                                                  | 
| **Название**              | Регистрация нового пользователя                                                                                                                                                                                                                                        |
| **Описание**              | Проверка возможности регистрации нового пользователя                                                                                                                                                                                                                   |  
| **Предусловия**           | Пользователь находится на главной странице сайта, пользователь с имейлом ivanov@example.com еще не зарегистрирован на сайте |     
| **Шаги**                  | 1. Перейти на вкладку "Регистрация".  <br>2. Заполнить все обязательные поля:  <br>- Имя и фамилия: "Иван Иванов" <br>- Email: "ivanov@example.com"  <br>- Пароль: "Password123"  <br>3. Отметить чекбокс о соглашении с обработкой персональных данных <br>4. Нажать кнопку "Зарегистрироваться". |  
| **Окружение** | Ноутбук ACER Aspire, Windows 11, GoogleChrome |
| **Ожидаемый результат**   | Пользователь успешно зарегистрирован и перенаправлен на страницу личного кабинета.                                                                                                                                                                                     |     
| **Фактический результат** |  Пользователь успешно зарегистрирован, отображается страница личного кабинета                                                                                                                                                                                          |     
| **Статус**                |                   Успех                                                                                                                                                                                                                                            |     
| **Приоритет**             | Высокий                                                                                                                                                                                                                                                               |
